## Getting Started

### Prerequisites

1. Get a Google Maps Platform API Key at [https://mapsplatform.google.com/](https://mapsplatform.google.com/)

2. Install numpy packages
   ```sh
   pip install numpy
   ```
3. Install openpyxl packages
   ```sh
   pip install openpyxl
   ```
4. Install pandas packages
   ```sh
   pip install pandas
   ```

   
### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Ryan-Pienaar/GeoAPI.git
   ```
   
2. Enter your Google Maps Platform API key in `main.py`
   ```py
   API_KEY = 'ENTER YOUR API';
   ```
